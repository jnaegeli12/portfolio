Washington state is only one of five states that allows its residents to vote by mail. Upon the input of an address, this application provides voters with ballot drop-box locations, candidates on the ballot, and news headlines related to the election.

The Weather Dashboard allows the user to enter any city and see today's weather as well as a 5-day forecast. The button click triggers an API call to openweathermap.org for temperatures, conditions, UV index, humidity, and icon indicators.
